# JavaFX Solitaire

* [Freecell](http://www.solitairecity.com/Help/FreeCell.shtml)
* [Bakers Dozen](http://www.lenagames.com/bp_files/rul/bakers-dozen.htm)
* [Ace's Up](http://www.lenagames.com/bp_files/rul/aces-up.htm)
